FitnessFormFixer

As the name may suggest, FitnessFormFixer is a web app that analyzes user form in a resistance training exercise.

With the integration of neural network and visualization tools, user can get real-time form adjustments as they do the exercise and form analyses once the session is ended.

GitHub repo

Recipeasy

Recipeasy is a recipe management web app, where user can look up, create, update, and delete a recipe.

In this collaboration, I helped implement various frontend functionalities, such as renders recipes on the homepage from the database, as well as the search and filtering features.

GitHub repo
